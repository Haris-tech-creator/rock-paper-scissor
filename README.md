# Rock Paper Scissor Game

This is a fun and interactive Rock Paper Scissor game developed using HTML, CSS, and JavaScript. 
Players can click buttons to choose Rock, Paper, or Scissors and play against the computer in real-time.

## 🎮 Features
- Interactive gameplay using buttons
- Dynamic score tracking for player and computer
- Modern and responsive UI
- Animations and visual feedback

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript

## 📂 Project Purpose
This project was developed to practice web development skills including DOM manipulation, event handling, and responsive design.
